## Cloud Security Application

This project implements a simple cloud security system using Java.

Features:
- Password Validation
- Authentication
- Role Based Access Control (RBAC)
- Data Encryption
- Security Logging

Workflow:
User Login → Password Validation → Access Control → Data Encryption → Log Monitoring
